# 3Ô∏è‚É£ STEP THREE: COPY THE CODE INTO EACH FILE

## üü¶ `js/songs.js`

This controls saga + song dropdowns.

```javascript
document.querySelectorAll(".saga-title").forEach(button => {
  button.addEventListener("click", () => {
    button.nextElementSibling.classList.toggle("open");
  });
});

document.querySelectorAll(".song").forEach(song => {
  song.addEventListener("click", () => {
    song.nextElementSibling.classList.toggle("open");
  });
});
```

Save the file.

---

## üîç `js/search.js`

This makes the search bar work.

```javascript
const searchInput = document.querySelector("#searchInput");

if (searchInput) {
  searchInput.addEventListener("keyup", function () {
    const filter = searchInput.value.toLowerCase();
    const items = document.querySelectorAll(".searchable");

    items.forEach(item => {
      const text = item.textContent.toLowerCase();
      item.style.display = text.includes(filter) ? "" : "none";
    });
  });
}
```

Save the file.
